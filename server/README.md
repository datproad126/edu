# Xử lý chèn text PDF và đọc text từ PDF
## description
- Cách xử lý: sử dụng 2 thư viện của js là pdf-lib và pdf-to-text để xử lý vấn đề
- Môi trường code: Nodejs, expressjs, typescript
- Mô hình: mvc

## installation
```
npm install
```
## Running dev
```
npm run dev
```
## deploy
```
npm run build && npm run start
```
